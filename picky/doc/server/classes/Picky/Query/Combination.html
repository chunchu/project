Class Picky::Query::Combination < Object

  • lib/picky/query/combination.rb

Describes the combination of a token (the text) and the index (the bundle): [text, index_bundle]

A combination is a single part of an allocation: […, [text2, index_bundle2], …]

An allocation consists of a number of combinations: [[text1, index_bundle1], [text2, index_bundle2], [text3, index_bundle1]]
